Objetivo Principal

Realizar el maquetado y diseño de una aplicación web basada en el framework de Angular en su versión 17 bajo el enfoque de código recursivo.

Objetivos Específicos

  • Utilización del framework de hojas de estilo denominado Tailwind.
  • Diseño de componentes recursivos.
  • Implementación de las directrices de Angular.
  • Formularios reactivos.
  • Interfaz adaptativa.
  • Disposición ordenada de archivos.
  • Consumo de una API.
  • Aplicación de buenas prácticas.
  • Comunicación entre componentes.
  • Utilización de rutas.
