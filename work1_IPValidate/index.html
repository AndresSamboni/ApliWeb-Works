Reto JavaScript: Control de entrada para direcciones IP

Enunciado del reto propuesto

Se quiere implementar un cuadro de texto (input de tipo text) que permita el ingreso de direcciones IP de manera interactiva aplicando las reglas de validación necesarias y suficientes para garantizar el formato adecuado de una dirección IP, mientras el usuario escribe. Se debe usar JavaScript puro o jQuery preferiblemente. Recuerden que la validación se debe realizar mientras el usuario esta escribiendo. Las reglas mínimas son:

  1. Primer carácter diferente de punto '.'.
  2. Máxima longitud de la cadena 15 caracteres, pero debe cambiar de acuerdo a los caracteres que se vayan digitado.
  3. Mínima longitud de cadena 7 caracteres.
  4. Máximo 3 caracteres de punto.
  5. Máximo 4 grupos de 3 dígitos
  6. Eliminar los ceros a la izquierda en los grupos de 3 dígitos.
  7. Valor numérico de cada grupo en el rango entre 0 y 255.
  8. No se permiten puntos consecutivos.
  9. Autocompletar el punto después de digitar 3 caracteres numéricos consecutivos.
  10. Solo se permiten caracteres numéricos y el punto, no se permiten letras u otros caracteres especiales.
  11. No se permite copiar y pegar en el cuadro de texto
